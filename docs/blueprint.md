# **App Name**: Task Ticker

## Core Features:

- Add Task: Input field for adding new tasks to the to-do list.
- Task Display: Display tasks with a checkbox and delete button.
- Task Completion: Checkbox to mark tasks as complete, with strikethrough effect.
- Task Deletion: Delete button to remove tasks from the list.
- Local Storage Persistence: Save and load task data from local storage for persistent state.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6) to promote focus and productivity.
- Background color: A very light blue (#F0F8FF), nearly white, providing a clean and unobtrusive backdrop.
- Accent color: A gentle green (#81C784) for completed tasks and positive interactions.
- Body and headline font: 'PT Sans', a humanist sans-serif offering a modern, readable style for all text.
